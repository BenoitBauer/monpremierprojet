Positions et Overflows

Les Positions

Introduction

La propriété css position permet de controler la manière dont un élément est possitioner dans le document.

Elle determine si un élément suit le flux normal du document ou s'il est placer precisement avec les propriétes top right, bottom, lefs

1. Position: Static;

Comportement: L'élément suit le flux normal, position par defaut de tous les élément html

Top, rigth, bottom, left n'on aucun effet si l'élément a cette propriété

Exemple

        
            .static {
                height: 50px;
                width: 50px;
                background-color: blue;
                position: static;
                }

JE SUIS COLLANT !!

2. Position Relative

Comportement: l'élément est positioné par rapport à sa position normal

Il occupe toujours sa place dans le flux du document

Les propriétées, top, right bottom et left, le déplace visuellement, sans changer sa place d'origine

        
           .relative {
                 height: 50px;
                 width: 50px;
                background-color: blue;
                position: relative;
                top: 2rem;
                left: 3rem;
                }

3. Position Absolue

L'élément est retirer du flux normal

Il est positionné par rapport au 1er parent positionné (non statique)

Si aucun parents n'est positionné, il se basera sur la balise html

4. Posistion fixe

L'élément fixé est retiré du fluxcomme absolute

Mais il est positionné par rapport à la fenetre de l'explorateur

Il ne bouge pas quand on scroll

Element fixé !!!

5. La position Sticky

Le comportement est un mixe entre relative et fixed

L'élément reste dans le flux jusqu'à ce qu'on scroll au point où il devient fixe

Le parent de l'élément sticky ne doit pas etre en overflow:hidden, sinon le sticky ne marche pas
